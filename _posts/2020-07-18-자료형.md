# 자료형

## ✔️ 정수 자료형

 char, short, int, long long

- char 자료형 : 1byte( = 8bit, 2^-7~ 2^7 )
    - unsigned char ( 0 ~ 2^8-1)
    - char ( -2^7 ~ 2^7-1)
- short : 2byte
- int : 4byte
- long long : 8byte

    80번째 피보나치 수를 구하는 문제와 같이 int가 표현할 수 있는 범위를 넘어서는 수를 저장해야 한다면 반드시 long long 사용해야함

## ✔️ 실수 자료형

 float, double

- float : 4byte
- double : 8byte
